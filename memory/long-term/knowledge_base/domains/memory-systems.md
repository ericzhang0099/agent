# 领域知识: 记忆系统设计

## 核心概念

### 三层记忆模型
基于人类记忆机制设计的三层架构:

1. **短期记忆 (Short-Term)**
   - 类似工作记忆
   - 容量有限，持续时间短
   - 存储当前活跃信息

2. **中期记忆 (Mid-Term)**
   - 类似情景记忆
   - 跨会话的项目状态
   - 有明确的生命周期

3. **长期记忆 (Long-Term)**
   - 类似语义记忆
   - 永久存储
   - 知识沉淀和经验总结

## 记忆流转原则

### 提取原则
- 重要性: 对用户有价值的信息
- 关联性: 与现有知识的关联
- 时效性: 当前或未来可能用到

### 遗忘原则
- 过期: 超过生命周期的短期记忆
- 完成: 已结束项目的相关记忆
- 过时: 被新信息替代的内容

## 最佳实践

1. **及时归档**: 会话结束后立即归档短期记忆
2. **定期整理**: 每周回顾中期记忆，更新长期记忆
3. **标签管理**: 使用统一标签体系便于检索
4. **版本控制**: 重要决策和知识保留历史版本
